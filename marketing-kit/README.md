# APS Drone Marketing Kit (DFW)

This folder contains practical assets to run weekly lead generation.

## Files
- `target-list-template.csv`: Prospecting sheet for DFW outreach.
- `email-templates.md`: Cold email and follow-up scripts by segment.
- `instagram-dm-templates.md`: DM scripts for warm social outreach.
- `review-request-templates.md`: Messages to request Google reviews after each job.
- `weekly-execution-checklist.md`: Weekly action plan to keep pipeline full.

## Core weekly cadence
1. Add 50 new prospects to target list.
2. Send 50 first-touch emails.
3. Send all scheduled follow-ups.
4. Send 5 review requests to recent clients.
5. Publish 1 Google Business Profile update with fresh media.
